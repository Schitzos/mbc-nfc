- Clean Architecture.
- Pure domain layer.
- NFC behind repository interface.
- Versioned card payload codec.
- Silent Shield isolated from UI and domain policies.
- Activity-based tariff and state policy.
- Station, Gate, Terminal, Scout as separate presentation role surfaces.
